💻 Entrada, Processamento e Saída

Conceitos

  • Entrada: dados recebidos pelo programa.
  • Processamento: operações realizadas com os dados.
  • Saída: resultado apresentado ao usuário.

Exemplo

idade = int(input("Digite sua idade: "))

idade_futura = idade + 10

print(idade_futura)

Entrada → idade
Processamento → idade + 10
Saída → resultado exibido

🔀 Condicionais (if, elif e else)

Permitem tomar decisões. 

idade = 18

if idade >= 18:
    print("Maior de idade")
else:
    print("Menor de idade")

Condicionais Alinhadas

nota = 8

if nota >= 9:
    print("Ótimo")
elif nota >= 7:
    print("Bom")
else:
    print("Reprovado")
📚 Vetores e Listas

Vetores (listas em Python) armazenam vários valores em uma única variável. 

nomes = ["Ana", "Carlos", "Pedro"]

print(nomes[0])

Resultado: Ana

Percorrendo Lista

for nome in nomes:
    print(nome)
🔄 Laços de Repetição

While

Executa enquanto a condição for verdadeira. 

contador = 1

while contador <= 5:
    print(contador)
    contador += 1

For

Utilizado quando sabemos quantas vezes repetir. 

for i in range(1, 6):
    print(i)
⚖️ Diferença entre While e For
While For
Quantidade indefinida Quantidade definida
Baseado em condição Baseado em sequência
Pode gerar loop infinito Mais seguro

Exemplo de Regra de Negócio
