📘 Função

Definição

Uma função é uma relação entre dois conjuntos em que cada elemento do domínio possui apenas uma imagem.

Ela transforma valores de entrada (x) em valores de saída (y).

Exemplo

f(x) = x + 3

x = 1 → y = 4
x = 2 → y = 5
x = 3 → y = 6

Cada valor de x possui apenas um valor correspondente de y.

📗 Domínio e Contradomínio

Domínio

É o conjunto dos valores de entrada da função. 

f(x) = x + 1

Domínio = {1, 2, 3}

Contradomínio

É o conjunto dos possíveis valores de saída. 

f(1) = 2
f(2) = 3
f(3) = 4

Contradomínio = {2, 3, 4}

Exemplo Completo

(A) x (B) y
1 3
2 4
3 5
  • Domínio = {1,2,3}
  • Contradomínio = {3,4,5}
  • Imagem = todos os valores gerados pela função. {3,4,5}
📈 Taxa de Variação

A taxa de variação mostra quanto o valor de y varia quando x varia.

Fórmula

          y₂ - y₁
a = ---------------
          x₂ - x₁
  • a = taxa de variação

Exemplo

A(1,2)
B(3,6)

a = (6 - 2) / (3 - 1)

a = 4 / 2

a = 2

Quando x aumenta 1 unidade, y aumenta 2 unidades.

Exemplo com Número Negativo

A(-2,1)
B(4,13)

a = (13 - 1) / (4 - (-2))

a = 12 / 6

a = 2

Lembre-se: menos com menos vira mais.

🎯 Intercepto

O intercepto é o ponto onde a reta cruza o eixo y. 

y = ax + b

a = taxa de variação
b = intercepto

Exemplo

y = 2x + 3

b = 3

Intercepto = (0,3)

Encontrando o Intercepto

Ponto (1,5)

y = 2x + b

5 = 2(1) + b

b = 5 - 2

b = 3

Equação final: y = 2x + 3

📊 Distribuição de Frequência

A distribuição de frequência organiza os dados em tabelas para facilitar a análise. 

Dados:

1, 2, 2, 3, 3, 4

Total = 6

Frequência Absoluta (fi)

Classe fi
1 1
2 2
3 2
4 1

Mostra quantas vezes cada valor aparece.

📉 Frequência Relativa (fr)

Fórmula

fr = fi / total

Exemplo

Classe 2:

fr = 2 / 6

fr = 0,33

33,3%

A soma das frequências relativas deve resultar em 100% (ou valor aproximado).

📚 Frequência Acumulada (Fi)
Classe fi Fi
1 1 1
2 2 3
3 2 5
4 1 6

O último valor de Fi sempre será igual ao total de dados.

📏 Mediana

A mediana é o valor central dos dados organizados em ordem crescente.

Quantidade Ímpar

2, 4, 5, 7, 9

Mediana = 5

Quantidade Par

1, 2, 2, 3, 3, 4

Valores centrais:
2 e 3

(2 + 3) / 2

Mediana = 2,5
